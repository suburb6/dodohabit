# DodoHabit Mobile App Guide

## Overview
DodoHabit is a comprehensive habit tracking mobile application designed to help users build, maintain, and visualize their daily routines. The app offers a modern, intuitive interface and a suite of features to support habit formation, progress tracking, and motivation.

---

## Key Features

### 1. Habit Tracking
- Create, edit, and delete habits
- Set daily, weekly, or custom schedules
- Mark habits as complete with a single tap
- View streaks and completion history

### 2. Visual Progress & Analytics
- Habit heatmaps for visualizing consistency
- Monthly and weekly progress grids
- Animated counters and progress widgets
- Streak tracking and longest streak display

### 3. Motivation & Rewards
- Streak share cards for social sharing
- Pro badge and upsell cards for premium features
- Motivational overlays and nudges

### 4. Customization
- Theme system with light/dark and custom themes
- Icon picker for habits
- Customizable widgets for home and lock screens

### 5. Data & Sync
- Cloud sync with secure authentication
- Data restore and backup options
- Integration with Google Sign-In and Health Connect

### 6. Widgets & Extensions
- Home screen widgets for quick habit check-ins
- iOS and Android widget support
- Widget design system for consistent UI

### 7. Social & Sharing
- Share progress and streaks with friends
- Social sign-in options

### 8. Accessibility & UX
- Error boundaries and robust error handling
- Responsive layouts for all device sizes
- Smooth animations and transitions

---

## Technical Highlights
- Built with React Native for cross-platform support
- Uses NativeWind and Tailwind CSS for styling
- Modular component architecture
- Context-based state management
- Integration with Firebase/Firestore for backend
- Custom native modules for health data

## Getting Started
1. Download the app from the App Store or Google Play
2. Sign in or create an account
3. Add your first habit and start tracking!
